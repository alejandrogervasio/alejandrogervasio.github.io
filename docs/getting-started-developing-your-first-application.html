Getting Started: Developing your First "Hello World" Java Application

Published by Alejandro Gervasio On March 24, 2018 Under Java SE

Introduction

Whether we already have a strong background on object-oriented programming, or just want to learn the paradigm’s key concepts with a minimal footprint, getting started programming in Java doesn't take very long, and it's pretty easy to learn even for beginners.

To get the ball rolling, let’s create a simple example: a naive application that displays in the console the classic "Hello World" message.

One little caveat before we get started: I assume that you have already installed Java on your computer. If that is not your case, please check the official documentation on how to do that for your operating system.

The "HelloWorld" Class

As we're going to develop the application without relying on any IDE in particular, launch your preferred text editor, e.g. NotePad on Windows or Text Edit on Mac, and type the following code:

public class HelloWorld {
                            
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

Now, save the code snippet as "HelloWorld.java" in the location of your choice on your computer. This is required, of course, as every source file in Java needs to have the ".java" extension appended to its name, so it can be compiled and executed later on, assuming that the file doesn't contain errors.

The Class, Step by Step

The above example is contrived, sure, but it shows in a nutshell how to create a basic Java program. Nevertheless, it's worth to analyze in detail each part so we can understand the role that plays each one of them:

  1. The "HelloWorld" class is defined as a public class by using the public access modifier and the class keyword, which means that the class can be accessed with no restrictions by any other class.
  2. The main method is, in this case, the only method implemented by the class, and also the entry point of the Java application. As we did before when we defined the class, the public access modifier is used to declare the method public and the void keyword to specify that it doesn't return a value.
  3. The System.out.println("Hello World!") line just prints out to the console the "Hello World" text and ends with a newline character.
